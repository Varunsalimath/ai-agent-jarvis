# Jarvis AI Agent 🤖

Jarvis is a simple AI agent chatbot built using Python.
It can perform tasks like solving math problems and searching the internet using tools.

## Features

* AI Agent reasoning
* Calculator tool
* Web search tool
* Local LLM using Ollama
* Streamlit chatbot interface

## Tech Stack

* Python
* LangChain
* Ollama (Llama3)
* Streamlit

## Setup Instructions

1. Clone the repository

git clone https://github.com/YOUR_USERNAME/ai-agent-jarvis.git

2. Navigate to the folder

cd ai-agent-jarvis

3. Create virtual environment

python -m venv venv

4. Activate environment

venv\Scripts\activate

5. Install dependencies

pip install -r requirements.txt

6. Run the AI model

ollama run llama3

7. Start the application

streamlit run app.py
